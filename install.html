🤖 Instalação Android
Opção A — APK direto (recomendado)
1
Baixe o gin-aromatics-lab.apk abaixo
2
Abra o arquivo no celular
3
Se bloqueado: Configurações → Segurança → Instalar apps desconhecidos → Permitir
4
Toque Instalar → ícone aparece na tela inicial
⬇ Baixar APK Android
Android 5.0+ · ~5MB
Opção B — Via Chrome (sem APK)
1
Abra o Chrome e acesse a URL acima
2
Aguarde ~5s — aparece banner "Instalar Gin Lab" na base da tela
3
Toque ⬇ Instalar
4
Ícone aparece na tela inicial — app abre sem barra do Chrome
🪟 Instalação Windows
1
Abra a URL no Chrome ou Edge
2
Chrome: clique no ícone ⊕ no canto direito da barra de endereços → Instalar
3
Edge: Menu ⋯ → Aplicativos → Instalar este site como aplicativo
4
O app abre em janela própria, aparece no menu Iniciar
🍎 Instalação iPhone / iPad
1
Abra a URL no Safari (não funciona no Chrome do iOS)
2
Toque no botão Compartilhar (□↑) na barra inferior
3
Role e toque em "Adicionar à Tela de Início"
4
Confirme o nome → toque Adicionar
⚠️ No iOS o modo offline é limitado pelo Safari. Para melhor experiência use Android ou desktop.
